Authentication System Styling

This project is a styled authentication system developed using Bootstrap 5 and custom CSS.
The main goal of this project is to convert basic HTML authentication pages into a modern, responsive, and visually appealing web application.

---

Pages Included

- Login Page
- Registration Page
- Forgot Password Page
- Reset Password Page
- Dashboard Page

Each page is designed using Bootstrap components such as cards, forms, buttons, and navbar to maintain a clean and professional layout.

---

Features

- Fully responsive design (works on mobile, tablet, and desktop)
- Bootstrap 5 integration for layout and styling
- Custom CSS for enhanced UI and design
- Password show/hide toggle functionality
- Password strength validation (uppercase, lowercase, number, special character)
- Dark mode toggle for better user experience
- Smooth hover effects and transitions

---

Technologies Used

- HTML
- CSS
- Bootstrap 5
- JavaScript
